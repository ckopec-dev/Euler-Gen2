# Euler Problem 34 Solution in Pascal

Euler Problem 34 asks: Find the sum of all numbers which are equal to the sum of the factorial of their digits.

Let me solve this step by step:

```pascal
program Euler34;

function Factorial(n: integer): integer;
var
  i, result: integer;
begin
  result := 1;
  for i := 1 to n do
    result := result * i;
  Factorial := result;
end;

function SumOfDigitFactorials(n: integer): integer;
var
  sum, digit: integer;
begin
  sum := 0;
  while n > 0 do
  begin
    digit := n mod 10;
    sum := sum + Factorial(digit);
    n := n div 10;
  end;
  SumOfDigitFactorials := sum;
end;

var
  i, totalSum: integer;
begin
  totalSum := 0;
  
  // We need to find an upper bound
  // For a number with d digits, if it equals sum of digit factorials
  // Then 10^(d-1) <= sum of factorials <= d * 9!
  // This gives us an upper limit for our search
  
  for i := 3 to 100000 do  // Reasonable upper bound
  begin
    if i = SumOfDigitFactorials(i) then
      totalSum := totalSum + i;
  end;
  
  writeln('Sum of all numbers equal to sum of factorial of digits: ', totalSum);
end.
```

## Explanation

1. **Factorial Function**: Computes the factorial of a given digit
2. **SumOfDigitFactorials Function**: Calculates the sum of factorials of all digits in a number
3. **Main Logic**: 
   - We iterate through numbers starting from 3 (since 1! = 1 and 2! = 2 are trivial cases)
   - For each number, we check if it equals the sum of factorials of its digits
   - We sum all such numbers

## Key Insights

- The upper bound can be estimated: for a d-digit number, we need 10^(d-1) ≤ d × 9!
- Since 9! = 362880, and 7 × 9! = 2540160, we only need to check up to about 10^7
- The valid numbers are: 145 (1! + 4! + 5! = 1 + 24 + 120 = 145) and 40585 (4! + 0! + 5! + 8! + 5! = 24 + 1 + 120 + 40320 + 120 = 40585)

## Result

The sum of all numbers which are equal to the sum of the factorial of their digits is **40730**.
