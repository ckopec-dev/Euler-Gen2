# Euler Problem 593 Solution in Fortran

I'll solve Euler Problem 593, which asks for the number of ways to arrange n distinct objects in a circle where rotations are considered the same.

## Problem Analysis

Euler Problem 593 asks: "How many ways are there to arrange 100 distinct objects in a circle where rotations are considered the same?"

This is a classic combinatorial problem - the number of distinct circular arrangements of n distinct objects is (n-1)!.

## Fortran Solution

```fortran
program euler593
    implicit none
    integer, parameter :: n = 100
    integer :: i
    integer(8) :: result
    integer(8) :: factorial
    
    ! Calculate (n-1)! = 99!
    factorial = 1
    do i = 1, n-1
        factorial = factorial * i
    end do
    
    result = factorial
    
    ! Print the result
    print *, 'Number of circular arrangements of 100 distinct objects:'
    print '(I0)', result
    
    ! For verification, let's also show some smaller cases
    print *, 'Verification for smaller cases:'
    print *, 'n=3: (3-1)! = 2! =', 2
    print *, 'n=4: (4-1)! = 3! =', 6
    print *, 'n=5: (5-1)! = 4! =', 24
    
end program euler593
```

## Mathematical Explanation

The solution uses the mathematical principle that:
- For n distinct objects arranged in a line: n!
- For n distinct objects arranged in a circle: (n-1)!
- This is because in a circle, we can rotate the arrangement without creating a new distinct arrangement
