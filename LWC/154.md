# Euler Problem 154 Solution in Lightning Web Component

I'll solve Euler Problem 154 using Lightning Web Components. This problem asks for the number of zeros at the end of 2000! (2000 factorial).

## Problem Understanding
We need to count how many trailing zeros are in 2000! This is equivalent to finding how many times 10 divides into 2000!, which means counting pairs of factors 2 and 5. Since there are always more factors of 2 than 5, we only need to count the factors of 5.

## Solution

```javascript
// lwc/euler154/Euler154.js
import { LightningElement } from 'lwc';

export default class Euler154 extends LightningElement {
    result = 0;
    calculationSteps = [];

    connectedCallback() {
        this.calculateTrailingZeros();
    }

    calculateTrailingZeros() {
        const n = 2000;
        let count = 0;
        let step = 1;
        
        // Add initial step
        this.calculationSteps.push({
            step: step++,
            description: `Finding trailing zeros in ${n}!`,
            value: 0
        });

        // Count factors of 5
        for (let i = 5; i <= n; i *= 5) {
            const contribution = Math.floor(n / i);
            count += contribution;
            
            this.calculationSteps.push({
                step: step++,
                description: `Adding factors from 5^${Math.log(i)/Math.log(5)}: ${contribution}`,
                value: count
            });
        }

        this.result = count;
        
        this.calculationSteps.push({
            step: step++,
            description: `Total trailing zeros in ${n}!`,
            value: count
        });
    }

    get displayResult() {
        return `The number of trailing zeros in 2000! is: ${this.result}`;
    }
}
```

## Algorithm Explanation

The solution uses Legendre's formula to count the number of trailing zeros in n!:

1. **Key Insight**: Trailing zeros are created by factors of 10, which equals 2 × 5
2. **Optimization**: Since there are always more factors of 2 than 5 in n!, we only count factors of 5
3. **Formula**: For n! we count:
   - Numbers divisible by 5: ⌊n/5⌋
   - Numbers divisible by 25 (5²): ⌊n/25⌋ (these contribute an extra factor of 5)
   - Numbers divisible by 125 (5³): ⌊n/125⌋ (these contribute two extra factors of 5)
   - And so on...

## Mathematical Calculation for 2000!

```
⌊2000/5⌋ = 400
⌊2000/25⌋ = 80
⌊2000/125⌋ = 16
⌊2000/625⌋ = 3
⌊2000/3125⌋ = 0
```

Total: 400 + 80 + 16 + 3 = 499 trailing zeros
