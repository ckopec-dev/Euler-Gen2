# Euler Problem 37 Solution in Lightning Web Component

Euler Problem 37 asks us to find the sum of the only eleven primes that are both truncatable from left to right and right to left.

Here's the solution implemented as a Lightning Web Component:

```javascript
// truncatablePrimes.js
import { LightningElement } from 'lwc';

export default class TruncatablePrimes extends LightningElement {
    result = 0;
    primes = [];
    truncatablePrimes = [];

    connectedCallback() {
        this.findTruncatablePrimes();
    }

    // Check if a number is prime
    isPrime(num) {
        if (num < 2) return false;
        if (num === 2) return true;
        if (num % 2 === 0) return false;
        
        for (let i = 3; i <= Math.sqrt(num); i += 2) {
            if (num % i === 0) return false;
        }
        return true;
    }

    // Check if a prime is truncatable from left to right
    isLeftTruncatable(prime) {
        const str = prime.toString();
        for (let i = 1; i < str.length; i++) {
            if (!this.isPrime(parseInt(str.substring(i)))) {
                return false;
            }
        }
        return true;
    }

    // Check if a prime is truncatable from right to left
    isRightTruncatable(prime) {
        const str = prime.toString();
        for (let i = str.length - 1; i > 0; i--) {
            if (!this.isPrime(parseInt(str.substring(0, i)))) {
                return false;
            }
        }
        return true;
    }

    // Find all truncatable primes
    findTruncatablePrimes() {
        const truncatablePrimes = [];
        let candidate = 11; // Start from 11 since 2, 3, 5, 7 are not truncatable
        
        while (truncatablePrimes.length < 11) {
            if (this.isPrime(candidate)) {
                if (this.isLeftTruncatable(candidate) && this.isRightTruncatable(candidate)) {
                    truncatablePrimes.push(candidate);
                }
            }
            candidate += 2; // Only check odd numbers
        }
        
        this.truncatablePrimes = truncatablePrimes;
        this.result = truncatablePrimes.reduce((sum, prime) => sum + prime, 0);
    }

    get truncatablePrimesList() {
        return this.truncatablePrimes.join(', ');
    }

    get sumResult() {
        return this.result;
    }
}
```

## Solution Explanation

This solution works by:

1. **Prime Checking**: The `isPrime()` method efficiently checks if a number is prime by testing divisibility up to the square root of the number.

2. **Truncation Checks**: 
   - `isLeftTruncatable()` checks if removing digits from the left still produces primes
   - `isRightTruncatable()` checks if removing digits from the right still produces primes

3. **Main Algorithm**: 
   - Start checking from 11 (smallest two-digit prime)
   - Only test odd numbers (since even numbers > 2 aren't prime)
   - For each prime, verify it's both left and right truncatable
   - Continue until we find 11 such primes

4. **Result**: The sum of the 11 truncatable primes is 748317

## Key Insights

- The solution only needs to check odd numbers since even numbers > 2 cannot be prime
- A truncatable prime must contain only odd digits (except for the single-digit primes 2 and 5, which are not truncatable)
- The algorithm efficiently combines prime checking with truncation validation

The final answer is **748317**.
