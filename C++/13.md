# Euler Problem 13 Solution in C++

## Problem Statement
Work out the first ten digits of the sum of the following one-hundred 50-digit numbers.

## Solution

```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    // Read the 50-digit numbers from file
    ifstream file("numbers.txt");
    vector<string> numbers;
    string line;
    
    while (getline(file, line)) {
        numbers.push_back(line);
    }
    file.close();
    
    // Perform addition digit by digit from right to left
    string result = "";
    int carry = 0;
    int numDigits = numbers[0].length();
    
    // Process each digit position from right to left
    for (int i = numDigits - 1; i >= 0; i--) {
        int sum = carry;
        
        // Add all numbers at current digit position
        for (int j = 0; j < numbers.size(); j++) {
            sum += (numbers[j][i] - '0');
        }
        
        // Calculate carry and current digit
        carry = sum / 10;
        result = char((sum % 10) + '0') + result;
    }
    
    // Handle remaining carry
    while (carry > 0) {
        result = char((carry % 10) + '0') + result;
        carry /= 10;
    }
    
    // Output first 10 digits
    cout << "First ten digits: " << result.substr(0, 10) << endl;
    
    return 0;
}
```

## Alternative Approach (More Efficient)

```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <vector>

using namespace std;

int main() {
    // Read numbers from file
    ifstream file("numbers.txt");
    vector<string> numbers;
    string line;
    
    while (getline(file, line)) {
        numbers.push_back(line);
    }
    file.close();
    
    // Use string addition to handle large numbers
    string sum = "0";
    
    for (const string& num : numbers) {
        // Add current number to sum
        int i = sum.length() - 1;
        int j = num.length() - 1;
        int carry = 0;
        string temp = "";
        
        while (i >= 0 || j >= 0 || carry > 0) {
            int digitSum = carry;
            
            if (i >= 0) {
                digitSum += (sum[i] - '0');
                i--;
            }
            
            if (j >= 0) {
                digitSum += (num[j] - '0');
                j--;
            }
            
            temp = char((digitSum % 10) + '0') + temp;
            carry = digitSum / 10;
        }
        
        sum = temp;
    }
    
    // Output first 10 digits
    cout << "First ten digits: " << sum.substr(0, 10) << endl;
    
    return 0;
}
```

## Key Points

1. **Large Number Handling**: Since each number is 50 digits long and we have 100 such numbers, the sum will be much larger than what standard integer types can handle.

2. **String-based Addition**: The solution uses string manipulation to perform addition digit by digit, similar to how we would do it manually.

3. **Right-to-Left Processing**: Addition is performed from the least significant digit (rightmost) to the most significant digit (leftmost).

4. **Carry Management**: Proper handling of carry values during addition ensures correct results.
