# Euler Problem 4 Solution in Ada

Euler Problem 4 asks: "Find the largest palindrome made from the product of two 3-digit numbers."

Here's my Ada solution:

```ada
with Ada.Text_IO; use Ada.Text_IO;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Euler_4 is
   Largest_Palindrome : Integer := 0;
   
   function Is_Palindrome(N : Integer) return Boolean is
      S : String(1..6);
      Length : Integer;
   begin
      -- Convert integer to string
      Length := 0;
      declare
         Temp : Integer := N;
      begin
         while Temp > 0 loop
            Length := Length + 1;
            S(Length) := Character'Val(Temp mod 10 + Character'Pos('0'));
            Temp := Temp / 10;
         end loop;
      end;
      
      -- Check if palindrome
      for I in 1..Length/2 loop
         if S(I) /= S(Length - I + 1) then
            return False;
         end if;
      end loop;
      
      return True;
   end Is_Palindrome;
   
begin
   -- Check all products of 3-digit numbers
   for I in 100..999 loop
      for J in I..999 loop  -- Start from I to avoid duplicate checks
         declare
            Product : Integer := I * J;
         begin
            if Is_Palindrome(Product) and Product > Largest_Palindrome then
               Largest_Palindrome := Product;
            end if;
         end;
      end loop;
   end loop;
   
   Put("The largest palindrome made from the product of two 3-digit numbers is: ");
   Put(Largest_Palindrome);
   New_Line;
end Euler_4;
```

## Explanation

1. **Main Logic**: The program iterates through all possible products of two 3-digit numbers (100-999).

2. **Palindrome Check**: The `Is_Palindrome` function converts an integer to a string and checks if it reads the same forwards and backwards.

3. **Optimization**: The inner loop starts from `I` instead of 100 to avoid checking the same product twice (e.g., 123×456 and 456×123).

4. **Result**: The program keeps track of the largest palindrome found and outputs it.
