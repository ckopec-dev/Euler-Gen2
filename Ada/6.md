# Euler Problem 6 Solution in Ada

## Problem Statement
The sum of the squares of the first ten natural numbers is,
1² + 2² + ... + 10² = 385

The square of the sum of the first ten natural numbers is,
(1 + 2 + ... + 10)² = 55² = 3025

Hence the difference between the sum of the squares of the first ten natural numbers and the square of the sum is 3025 − 385 = 2640.

Find the difference between the sum of the squares of the first one hundred natural numbers and the square of the sum.

## Solution

```ada
with Ada.Text_IO; use Ada.Text_IO;

procedure Euler_006 is
   N : constant := 100;
   
   -- Calculate sum of squares: 1² + 2² + ... + N²
   Sum_Of_Squares : Long_Long_Integer := 0;
   
   -- Calculate square of sum: (1 + 2 + ... + N)²
   Square_Of_Sum : Long_Long_Integer := 0;
   Sum : Long_Long_Integer := 0;
   
begin
   -- Calculate sum of squares
   for I in 1 .. N loop
      Sum_Of_Squares := Sum_Of_Squares + Long_Long_Integer(I) ** 2;
   end loop;
   
   -- Calculate square of sum
   for I in 1 .. N loop
      Sum := Sum + Long_Long_Integer(I);
   end loop;
   
   Square_Of_Sum := Sum ** 2;
   
   -- Calculate the difference
   declare
      Difference : Long_Long_Integer := Square_Of_Sum - Sum_Of_Squares;
   begin
      Put_Line("The difference is: " & Long_Long_Integer'Image(Difference));
   end;
end Euler_006;
```

## Alternative Mathematical Approach

```ada
with Ada.Text_IO; use Ada.Text_IO;

procedure Euler_006_Math is
   N : constant := 100;
   
   -- Using mathematical formulas:
   -- Sum of first N natural numbers = N(N+1)/2
   -- Sum of squares of first N natural numbers = N(N+1)(2N+1)/6
   
   Sum_Of_N : Long_Long_Integer := N * (N + 1) / 2;
   Sum_Of_Squares : Long_Long_Integer := N * (N + 1) * (2 * N + 1) / 6;
   
   Square_Of_Sum : Long_Long_Integer := Sum_Of_N ** 2;
   Difference : Long_Long_Integer := Square_Of_Sum - Sum_Of_Squares;
   
begin
   Put_Line("The difference is: " & Long_Long_Integer'Image(Difference));
end Euler_006_Math;
```

## Explanation

The problem asks for the difference between:
1. The square of the sum of the first 100 natural numbers: (1 + 2 + ... + 100)²
2. The sum of the squares of the first 100 natural numbers: 1² + 2² + ... + 100²

**Method 1** uses straightforward iteration to calculate both values and then finds their difference.

**Method 2** uses mathematical formulas for efficiency:
- Sum of first N natural numbers: N(N+1)/2
- Sum of squares of first N natural numbers: N(N+1)(2N+1)/6

Both methods yield the same result: **25164150**
