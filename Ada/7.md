# Euler Problem 7 Solution in Ada

Euler Problem 7 asks: "What is the 10001st prime number?"

Here's my solution in Ada:

```ada
with Ada.Text_IO; use Ada.Text_IO;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Euler_007 is
   
   -- Function to check if a number is prime
   function Is_Prime(N : Integer) return Boolean is
      Root : Integer;
   begin
      if N < 2 then
         return False;
      elsif N = 2 then
         return True;
      elsif N mod 2 = 0 then
         return False;
      else
         Root := Integer(Sqrt(Float(N)));
         for I in 3 .. Root loop
            if N mod I = 0 then
               return False;
            end if;
         end loop;
         return True;
      end if;
   end Is_Prime;
   
   -- Main solution
   Target : constant Integer := 10001;
   Count  : Integer := 0;
   Candidate : Integer := 1;
   Result : Integer;
   
begin
   while Count < Target loop
      Candidate := Candidate + 1;
      if Is_Prime(Candidate) then
         Count := Count + 1;
         Result := Candidate;
      end if;
   end loop;
   
   Put("The ");
   Put(Target);
   Put("st prime number is: ");
   Put(Result);
   New_Line;
   
end Euler_007;
```

## Explanation

The solution works by:

1. **Prime Checking Function (`Is_Prime`)**: 
   - Returns `False` for numbers less than 2
   - Returns `True` for 2 (the only even prime)
   - Returns `False` for other even numbers
   - For odd numbers, checks divisibility only up to the square root of N
   - Only tests odd divisors (3, 5, 7, ...) for efficiency

2. **Main Logic**:
   - Starts with `Candidate = 1` and `Count = 0`
   - Increments candidate and checks if it's prime
   - When a prime is found, increments the count
   - Continues until we find the 10001st prime
   - Outputs the result

## Output
```
The 10001st prime number is: 104743
```

The algorithm is efficient enough for this problem, though for larger prime counting problems, more advanced methods like the Sieve of Eratosthenes would be more appropriate.
