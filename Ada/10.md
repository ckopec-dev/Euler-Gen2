# Euler Problem 10 Solution in Ada

## Problem Statement
The sum of the primes below 10 is 2 + 3 + 5 + 7 = 17.
Find the sum of all the primes below two million.

## Solution

```ada
with Ada.Text_IO; use Ada.Text_IO;
with Ada.Long_Long_Integer; use Ada.Long_Long_Integer;

procedure Euler_10 is
   
   Limit : constant := 2_000_000;
   
   -- Function to check if a number is prime
   function Is_Prime(N : Long_Long_Integer) return Boolean is
      Root : Long_Long_Integer;
   begin
      if N < 2 then
         return False;
      elsif N = 2 then
         return True;
      elsif N mod 2 = 0 then
         return False;
      else
         Root := Long_Long_Integer(Real(Sqrt(Real(N))));
         for I in 3 .. Root loop
            if N mod I = 0 then
               return False;
            end if;
         end loop;
         return True;
      end if;
   end Is_Prime;
   
   Sum : Long_Long_Integer := 0;
   I   : Long_Long_Integer := 2;
   
begin
   while I < Limit loop
      if Is_Prime(I) then
         Sum := Sum + I;
      end if;
      I := I + 1;
   end loop;
   
   Put_Line("Sum of primes below " & Limit'Img & " is: " & Sum'Img);
end Euler_10;
```

## Explanation

This solution works by:

1. **Prime Checking Function**: The `Is_Prime` function efficiently determines if a number is prime by:
   - Handling special cases (numbers less than 2, and the number 2)
   - Eliminating even numbers greater than 2
   - Testing divisibility only up to the square root of the number

2. **Main Loop**: Iterates through all numbers from 2 to 1,999,999, checking each for primality

3. **Summation**: Adds each prime number to the running total

4. **Data Types**: Uses `Long_Long_Integer` to handle the large sum that exceeds the range of standard integer types
